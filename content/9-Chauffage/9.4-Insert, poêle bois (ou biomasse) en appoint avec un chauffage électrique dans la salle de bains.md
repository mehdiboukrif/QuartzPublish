# 9.4-Installation de chauffage par insert, poêle bois (ou biomasse) avec un chauffage électrique dans la salle de bains

Dans cette configuration, valable que pour les maisons individuelles, tout le bâtiment est chauffé par un poêle bois. Seule la salle de bains est chauffée par un système électrique.

La consommation annuelle de chauffage Cch1 liée au poêle bois (kWh PCI$)$ est donnée par la formule :

$$
\text { Cch } 1=0,9 * \text { Bch } * I N T 1 * I \operatorname{ch} 1
$$

La consommation annuelle de chauffage Cch2 liée au chauffage électrique de la salle de bains (kWh PCI) est donnée par la formule :

$$
\text { Cch } 2=0,1 * \text { Bch } * I N T 2 * \text { Ich } 2
$$

L'émetteur poêle bois ou insert est traité ici comme un émetteur de base. Le chauffage électrique dans la salle de bain est saisi comme un émetteur de SDB. En présence de plusieurs salles de bain avec un chauffage électrique différent, la part de la consommation apportée par l'appoint est répartie entre les deux équipements par un prorata de surface habitable. C'est-à-dire pour le cas d'un logement avec deux salles de bains de surface $\mathrm{Sh}_{\text {sdb1 }}$ et $\mathrm{Sh}_{\text {sdb2 }}$ :

$$
\begin{aligned}
& C c h 2_{s b d 1}=0,1 * \frac{S h_{s d b 1}}{S h_{s d b 1}+S h_{s d b 2}} * B c h * I N T_{s d b 1} * I c h_{s d b 1} \\
& C c h 2_{s b d 2}=0,1 * \frac{S h_{s d b 2}}{S h_{s d b 1}+S h_{s d b 2}} * B c h * I N T_{s d b 2} * I c h_{s d b 2}
\end{aligned}
$$

Avec :
$$
C c h 2=C c h 2_{s d b 1}+C c h 2_{s d b 2}
$$
